WiseSpine is an AI pipeline for automated Cobb angle measurement from spine X-rays using three deep learning segmentation models across 481 annotated images.

## Key Contributions

- Built spine reconstruction pipeline for automated Cobb angle measurement
- Developed segmentation benchmarking system with spatial analysis and quantitative metrics
- Collaborated with AWS engineers on scalable AI infrastructure design
